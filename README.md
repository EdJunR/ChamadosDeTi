# ChamadosDeTi
Esse é meu primeiro projeto solo e pretendo conectá-lo com PHP.

O projeto necessita ter, para cada chamado:

Id, Titulo, Data de Abertura, Data de Atualização, Status, anexos da abertura, e-mail do solicitiante, *email dos demais envolvidos*, **histórico de alterações**.

### Histórico de Alterações:
Id, Data da atualização, anexos comprovantes da atualização *obrigatório*
